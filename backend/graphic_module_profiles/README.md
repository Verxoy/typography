# ICC profiles for CMYK conversion

Place CMYK ICC profiles from the [Graphics_Module](https://github.com/Nagibator2999/Graphics_Module) repository here:

- `coated.icc` — coated paper (ISO Coated)
- `uncoated.icc` — uncoated paper

Without these files the module uses a mathematical RGB→CMYK fallback (less accurate for print).

Copy from:

`Graphics_Module/color-converter-backend/profiles/`
